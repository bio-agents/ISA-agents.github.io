MetaboLights: towards a new COSMOS of metabolomics data management http://isa-agents.org/2012/09/new-paper-metabolights-towards-a-new-cosmos-of-metabolomics-data-management/ http://isa-agents.org/2012/09/new-paper-metabolights-towards-a-new-cosmos-of-metabolomics-data-management/#comments Thu, 27 Sep 2012 10:13:38 +0000 http://isaagents.wordpress.com/?p=410 A new paper, featuring the ISA team as authors is now available to read on MetaboLights (EMBL-EBI metabolomics repository).

MetaboLights, part of the ISA commons is the new European repository for metabolomic data held at the EBI. Their infrastructure is built upon the ISA software stack, utilising ISAcreator for data entry, ISA-Tab as the format then the BII data manager and database for persistence and storage respectively.
